# VagrantMCServer

Simple vagrant file to automatically install a minecraft server on Rocky 9. 

1. Clone this repo
2. Ensure Vagrant is installed

    `vagrant --version`

3. Start/Provision the envrionment
    
    `vagrant up`

4. Connect the the VM

    `vagrant ssh`

5. Start the server

    `cd mc`
    
    `./startup.sh`
